String 类型

1.’’ 或者 “” 都可以表示一个字符串

  1. 字符字面量，就是转义字符
字面量 含义
\n 换行
\t 制表符
\b 空格
\r 回车
\f 进纸
\\ 斜杠
\‘ 单引号，如 ‘ It \‘s mine ‘
\“ 双引号， 如 “ It \“s mine “
\xnn 以十六进制(n为0~f)表示的字符，如’\x41’表示 ‘A’
\unnnn 以十六进制(n为0~f)表示Unicode字符，如’\x03a3’表示希腊字母Σ
